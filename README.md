# PROJETO DE EXTENSÃO - ENADE

## 📌 Sobre o Projeto

Este repositório contém a documentação de requisitos e modelagem de um sistema voltado para a preparação de estudantes para o **Exame Nacional de Desempenho dos Estudantes (ENADE)**. O projeto é desenvolvido como parte da disciplina de **Requisitos e Modelagem de Sistemas** da **Unifor.**
